# Cascade Health & Energy Systems — GRC Compliance Program

A 7-tab Excel portfolio project demonstrating hands-on GRC analyst skills 
targeted at healthcare cybersecurity roles.

## What's Inside

| Tab | Content |
|---|---|
| Company Profile | Fictional org scope, assets, regulatory obligations |
| Vendor Assessment | Third party risk log with risk scoring and color coding |
| Control Matrix IT | HIPAA + ISO 27001 control assessment with gap identification |
| Control Matrix OT | NIST 800-82 + IEC 62443 OT/SCADA control assessment |
| Risk Register | 8 risks scored by Likelihood x Impact with mitigation owners |
| Policy Exception Log | Documented exceptions with approval dates and expiry tracking |
| Remediation Roadmap | Prioritized gap-to-action plan with timelines and status |

## Frameworks Covered
- HIPAA Security Rule
- ISO 27001 Annex A
- NIST SP 800-82 (ICS/SCADA)
- IEC 62443
